# 类人脑双系统全闭环 AI 架构 - 依赖包

## ⚠️ 重要提示

**Python 版本要求**: 必须使用 **Python3.11** (PyTorch 兼容性)

```bash
# 检查 Python 版本
python --version  # 应该是 3.11.x

# 如果版本不对，使用 conda 创建正确环境
conda create -n stdpbrain python=3.11
conda activate stdpbrain
```

## 核心依赖（必需）

### PyTorch
```bash
# CPU 版本（推荐用于测试）
conda install pytorch cpuonly -c pytorch

# GPU 版本（需要 CUDA）
conda install pytorch torchvision cudatoolkit -c pytorch
```

### Transformers 与模型加载
```bash
pip install transformers sentencepiece accelerate optimum
```

### 基础工具
```bash
pip install numpy scipy scikit-learn pandas tqdm pyyaml
```

### Telegram Bot（可选但推荐）
```bash
pip install python-telegram-bot aiohttp
```

## 一键安装脚本

### 方法 1: 使用 Conda（推荐）

```bash
#!/bin/bash
# setup_conda_env.sh

# 创建环境
conda create -n stdpbrain python=3.11 -y
conda activate stdpbrain

# 安装 PyTorch (CPU)
conda install pytorch cpuonly -c pytorch -y

# 安装其他依赖
pip install transformers sentencepiece accelerate optimum
pip install python-telegram-bot aiohttp
pip install numpy scipy scikit-learn pandas tqdm pyyaml

echo "✅ 环境配置完成!"
echo "使用方法：conda activate stdpbrain"
```

运行：
```bash
chmod +x setup_conda_env.sh
./setup_conda_env.sh
```

### 方法 2: 使用 Pip

```bash
# 确保 Python3.11
python3.11-m venv venv
source venv/bin/activate

# 安装 PyTorch
pip install torch --index-url https://download.pytorch.org/whl/cpu

# 安装其他依赖
pip install -r requirements.txt
```

## 常见问题

### Q1: Python 版本不对
```bash
# 错误：ModuleNotFoundError: No module named 'torch'
# 解决：使用 Python3.11
conda create -n stdpbrain python=3.11
```

### Q2: NumPy 版本冲突
```bash
# 错误：ValueError: numpy.dtype size changed
# 解决：降级 NumPy
pip install "numpy<2.0"
```

### Q3: Transformers 版本太低
```bash
# 错误：Qwen3.5 not supported
# 解决：升级 Transformers
pip install --upgrade transformers
```
