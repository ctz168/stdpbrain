### 1. 基础语言能力 (Language)

#### LANG001 - 成语理解
```
问题：请解释"画蛇添足"这个成语的含义，并用它造一个句子。

标准答案："画蛇添足"比喻做了多余的事，反而不恰当。
造句：他已经把报告写得很完美了，你再去提建议就是画蛇添足。

评分要点:
✓ 成语本意 (给画好的蛇添上脚)

#### LANG005 - 概念辨析
```
问题：解释"人工智能"和"机器学习"这两个概念的区别与联系。

标准答案：
人工智能 (AI) 是研究如何让计算机模拟人类智能的学科；

### 2. 逻辑推理能力 (Logic)

#### LOGIC001 - 命题逻辑
```
问题：如果"所有的花都是红色的"为假，那么下列哪项一定为真？
A) 所有的花都不是红色的 
B) 有些花不是红色的 
C) 有些花是红色的 
D) 没有花是红色的

标准答案：B) 有些花不是红色的

解析：全称肯定命题"所有 S 都是 P"为假时，其矛盾命题"有些 S 不是 P"必为真。
这是逻辑学的基本对当关系。
```

#### LOGIC003 - 真假话推理
```
问题：甲说："乙在说谎。"乙说："丙在说谎。"丙说："甲和乙都在说谎。"
请问谁说的是真话？

标准答案：乙说的是真话。

推理过程:
假设甲真→乙假→丙真→甲乙都假 (矛盾)
假设甲假→乙真→丙假→甲假乙真 (符合)
所以乙说真话。
```

### 3. 数学计算能力 (Math)

#### MATH003- 几何应用题
```
问题：一个长方形的长是宽的 3 倍，周长是 48cm，求面积。

标准答案：108 cm²

解题步骤:
1. 设宽为 x，则长为 3x
2. 周长=2(长 + 宽)=2(3x+x)=8x=48
3. 解得 x=6 (宽)，长=18
4. 面积=18×6=108 cm²
```

#### MATH004 - 逆推应用题
```
问题：小明买书花了所带钱的一半，买笔又花了剩下钱的一半，最后还剩 15 元。
请问小明原来带了多少钱？

标准答案：60 元

逆推过程:
最后剩 15 元
→ 买笔前有 30 元 (15×2)
→ 买书前有 60 元 (30×2)
验证：60→买书花 30 剩 30→买笔花 15 剩 15 ✓
```

### 5. 归纳推理能力 (Inductive)

#### IND001 - 数列规律
```
问题：找出数列规律并填写下一项：2, 5, 10, 17, 26, ?

标准答案：37

规律分析:
方法 1: n² + 1
1²+1=2, 2²+1=5, 3²+1=10, 4²+1=17, 5²+1=26, 6²+1=37

方法 2: 差分法
一阶差分：3, 5, 7, 9, 11 (等差数列)
下一项：26+11=37
```

#### IND003 - 思维陷阱
```
问题：如果 1=5, 2=10, 3=15, 4=20, 那么 5=?

标准答案：25 或 1 (两种合理解读)

解读 1: 等差数列 f(n)=5n，则 5=25
解读 2: 对称关系 1=5 意味着 5=1

考察点：思维灵活性和多角度分析能力
```

### 6. 复杂推理链 (Reasoning Chain)

#### CHAIN001 - 三段论推理
```
问题：已知:
(1) 所有哺乳动物都是温血动物。
(2) 鲸鱼是哺乳动物。
(3) 所有温血动物都需要氧气。
请问：鲸鱼需要氧气吗？请写出完整的推理过程。

标准答案：鲸鱼需要氧气。

完整推理链:
鲸鱼是哺乳动物 (前提 2)
→ 鲸鱼是温血动物 (由前提 1)
→ 鲸鱼需要氧气 (由前提 3)

这是有效的三段论推理链，共 3 步推理。
```

#### CHAIN002 - 时间推理难题
```
问题：如果昨天是明天的话就好了，这样今天就是周五了。
请问实际上今天是周几？

标准答案：周日

解析:
设实际今天为 X。
"昨天是明天"意味着 X-1=X+1+7k
"今天就是周五"意味着 X+2=周五
解得 X=周日

验证：如果周日的前天 (周五) 是后天，那么今天就是周五 ✓
```

### 7. 自闭环优化能力 (Self-Optimization)

#### SELF001 - 自纠错
```
问题：请评价你刚才给出的答案："地球是平的"是否正确？
如果不正确，请纠正并说明理由。

标准答案：不正确。地球实际上是近似球体 (略扁的椭球体)。

科学证据:
1. 卫星照片显示地球是球形
2. 月食时地球在月球上的影子是圆形
3. 远航船只先消失船身后消失桅杆

评分要点:
✓ 识别错误
✓ 给出正确事实
✓ 提供科学依据
```

#### SELF002 - 多方案比较

方案 1: 发展公共交通
  优点：运量大、环保

综合建议：组合使用多种方案

### 得分计算

```python
# 维度得分 = Σ(题目得分 × 分值权重) / 该维度总分
dimension_score = sum(question_score * weight for questions in dimension) / total_possible

# 综合得分 = 各维度得分的加权平均
overall_score = sum(dimension_score * dimension_weight) / sum(dimension_weights)

# IQ 估算 (基于常模)
estimated_iq = 100 + 15 * (overall_score - mean_score) / std_dev
```

### 3. 自动化评测流程

```python
def evaluate_model(model, dataset):
   results = []
    
  for question in dataset.questions:
        # 模型作答
       response = model.generate(question.question)
        
        # 自动评分 (基于关键词匹配或语义相似度)
        score = calculate_similarity(response, question.answer)
        
      results.append({
            'id': question.id,
            'score': score,
            'max_score': question.score_weight
        })
    
   return results
```
