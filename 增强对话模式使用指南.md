# 增强对话模式使用指南

## 🎯 功能特性

增强版对话模式参考了Telegram Bot的实现，提供了更丰富的交互体验：

### ✨ 核心特性

1. **实时内心独白**
   - AI会在对话间隙自发思考
   - 实时显示AI的思考过程
   - 类似Bot模式的异步独白流

2. **异步思考架构**
   - 后台独白生成线程
   - 主线程处理用户输入
   - 多线程协同工作

3. **丰富的交互命令**
   - `quit` / `exit` - 退出程序
   - `think` - 触发AI主动思考
   - `history` - 查看对话历史

4. **彩色输出**
   - 🟢 绿色：用户输入
   - 🟡 黄色：AI回复
   - 🔵 蓝色：系统状态
   - ⚫ 灰色：内心独白

## 🚀 使用方法

### 启动增强模式（默认）

```powershell
# 方式1：双击运行
对话模式.bat

# 方式2：命令行
venv\Scripts\activate
python main.py --mode chat
```

### 启动简单模式

```powershell
# 方式1：双击运行
简单对话模式.bat

# 方式2：命令行
python main.py --mode chat --simple
```

## ⚙️ 配置选项

### 命令行参数

```powershell
# 增强模式（默认）
python main.py --mode chat

# 简单模式
python main.py --mode chat --simple

# 指定设备
python main.py --mode chat --device cpu

# 指定量化
python main.py --mode chat --quantization FP16
```

### 高级配置

编辑 `enhanced_chat.py` 可以调整：

- `random.randint(30, 60)` - 独白生成间隔（秒）
- `max_tokens=200` - 回复最大长度
- `max_tokens=100` - 独白最大长度

## 🔄 工作原理

### 架构设计

```
┌─────────────────────────────────────────┐
│          主显示循环                      │
│  - 检查用户输入                          │
│  - 显示独白输出                          │
│  - 显示AI回复                            │
└──────────────┬──────────────────────────┘
               │
       ┌───────┴────────┐
       ▼                ▼
┌─────────────┐  ┌──────────────┐
│ 输入线程     │  │ 独白生成线程  │
│ - 异步输入   │  │ - 后台思考    │
│ - 命令处理   │  │ - 自发独白    │
└─────────────┘  └──────────────┘
```

### 线程说明

1. **主线程**
   - 显示独白和响应
   - 协调各线程工作
   - 处理特殊命令

2. **输入线程**
   - 异步接收用户输入
   - 不阻塞主显示循环

3. **独白线程**
   - 后台持续运行
   - 定期生成内心独白
   - 模拟AI自主思考

## 🐛 故障排除

### Q: 独白不显示？
**A:**
- 检查AI接口是否支持 `think()` 或 `generate_monologue_stream()`
- 查看控制台错误信息
- 尝试使用简单模式

### Q: 输入卡顿？
**A:**
- 这是正常的，因为模型推理需要时间
- CPU模式下约2-5秒
- GPU模式下约0.5-1秒

### Q: 颜色显示异常？
**A:**
- 确保终端支持ANSI颜色码
- Windows Terminal / PowerShell 支持
- CMD可能需要额外配置
