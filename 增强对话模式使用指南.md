# 增强对话模式使用指南

## 🎯 功能特性

增强版对话模式参考了Telegram Bot的实现，提供了更丰富的交互体验：

### ✨ 核心特性

1. **实时内心独白流**
   - AI会在回复前显示思考过程（潜意识）
   - 流式显示独白内容
   - 独白内容更像人脑思考：碎片化、自然跳跃
   - 支持思维状态机（专注、漂移、反思、休息）

2. **优化的独白生成**
   - 基于思维状态机的自然独白
   - 碎片化思维表达（不是完整句子）
   - 思维联想和跳跃
   - 更像人类的内心独白

3. **流式回复输出**
   - 逐字符流式输出
   - 实时显示生成过程
   - 提升交互体验

4. **彩色输出**
   - 🟢 绿色：用户输入
   - 🟡 黄色：AI回复
   - 🔵 蓝色：系统状态
   - ⚫ 灰色：内心独白

## 🚀 使用方法

### 启动增强模式（默认）

```powershell

对话模式.bat

# 方式2：命令行
venv\Scripts\activate
```

```powershell

# 方式2：命令行

```

## 🔄 工作原理

### 独白生成流程

```
用户输入
    ↓
设置思维种子
    ↓
召回海马体记忆
    ↓
生成潜意识独白
    ├─ 状态判断（专注/漂移/反思/休息）
    ├─ 构建自然思维Prompt
    ├─ 模型生成独白
    └─ 清理和优化独白内容
    ↓
流式显示独白
    ↓
生成正式回复
    ↓
存储记忆 & STDP学习
```

### 思维状态机

独白引擎实现了类人脑的思维状态机：

1. **专注模式 (FOCUSED)**
   - 围绕主题深入思考
   - 响应用户输入
   - 分析问题

2. **漂移模式 (WANDERING)**
   - 自由联想
   - 思维跳跃
   - 记忆关联

3. **反思模式 (REFLECTING)**
   - 自我审视
   - 回顾刚才的想法
   - 元认知能力

4. **休息模式 (RESTING)**
   - 后台处理
   - 沉默思考
   - 记忆整理

### 独白优化特点

1. **碎片化思维**
   - 不是完整句子
   - 省略主语谓语
   - 自然停顿

2. **思维跳跃**
   - 联想链生成
   - 主题漂移和回归
   - 自然过渡

3. **情绪影响**
   - 好奇、困惑、兴奋、平静、沉思
   - 情绪状态影响思考风格

4. **记忆关联**
   - 海马体记忆召回
   - 基于记忆的联想
   - 时间线索

## 🐛 故障排除

### Q: 独白不显示？
**A:**
- 检查AI接口是否支持 `chat_stream()` 方法
- 查看控制台错误信息
- 系统会自动降级到基础对话模式

### Q: 独白内容很奇怪？
**A:**
- 这是正常的，独白模拟的是碎片化思维
- 独白不是完整句子，而是思维片段
- 如果出现大量特殊符号，可能是模型问题

### Q: 输入卡顿？
**A:**
- 这是正常的，模型推理需要时间
- CPU模式下约2-5秒
- GPU模式下约0.5-1秒
- 独白生成会增加约0.5秒

### Q: 颜色显示异常？
**A:**
- 确保终端支持ANSI颜色码
- Windows Terminal / PowerShell 支持
- CMD可能需要额外配置

## 📝 技术细节

### 核心文件

- **`main.py`**: 增强对话模式入口
- **`core/interfaces.py`**: AI接口实现（`chat_stream` 方法）
- **`core/monologue_engine.py`**: 独白引擎（思维状态机）
- **`telegram_bot/bot.py`**: Telegram Bot实现（参考）

### 独白生成参数

```python
# 独白长度
max_tokens=35  # 足够表达思维片段

# 温度参数
temperature=0.85  # 较高温度增加多样性

# 停止条件
stop_token_ids=[151645, 151643]  # <|im_end|>, eos
```

### 性能优化

1. **异步生成**
   - 使用 `asyncio.to_thread` 避免阻塞
   - 流式输出提升响应感

2. **记忆召回优化**
   - 仅召回top2相关记忆
   - 使用embedding快速检索

3. **独白缓存**
   - 保持最近10条独白历史
   - 用于思维延续

### 依赖

- `asyncio` - 异步支持
- `torch` - 模型推理
- `transformers` - Qwen模型
- `datetime` - 时间戳

### 内存占用

- 基础：~2GB（模型）
- 独白引擎额外：~50MB
- 海马体记忆：~100MB
