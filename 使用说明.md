# 类人脑双系统全闭环 AI架构

## 🚀 快速开始（国内用户）

### 方式一：一键部署（推荐）

双击运行 **`一键部署.bat`**，脚本会自动完成：
- ✅ 检查Python环境
- ✅ 创建虚拟环境
- ✅ 安装所有依赖（使用清华镜像）
- ✅ 下载Qwen模型（使用ModelScope国内镜像）
- ✅ 启动程序

### 方式二：手动部署

```powershell
# 1. 创建虚拟环境
python -m venv venv

# 2. 激活虚拟环境
venv\Scripts\activate

# 3. 配置pip镜像源（加速下载）
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# 4. 安装依赖
pip install torch transformers huggingface_hub numpy accelerate
pip install python-telegram-bot aiohttp pydantic python-dotenv loguru modelscope

# 5. 下载模型（ModelScope国内镜像）
python -c "from modelscope import snapshot_download; snapshot_download('Qwen/Qwen3.5-0.8B', cache_dir='models')"

# 6. 移动模型文件
xcopy /E /I /Y models\Qwen\Qwen* models\Qwen3.5-0.8B

# 7. 运行
python main.py --mode chat
```

## ❓ 常见问题

### Q: 提示"python不是内部或外部命令"
**A:**
1. 确认Python已安装
2. 安装时勾选了 "Add Python to PATH"
3. 重新打开命令行窗口

### Q: 依赖安装速度慢
**A:** 已自动配置清华镜像源，如需更换可编辑上述镜像列表

### Q: 模型下载失败
**A:**
1. 检查网络连接
2. 手动下载：访问 https://modelscope.cn/models/Qwen/Qwen3.5-0.8B/files
3. 下载所有文件到 `models\Qwen3.5-0.8B\` 目录

### Q: 程序运行卡顿
**A:** CPU模式下模型推理较慢，属于正常现象。如有GPU可配置使用CUDA加速。
