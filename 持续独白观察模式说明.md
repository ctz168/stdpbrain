# 持续独白观察模式使用指南

## 🎯 功能特性

持续独白观察模式让你可以实时观察AI的思维流，就像观察一个真实人脑的内心活动一样。

### ✨ 核心特性

1. **持续思维流**
   - AI会在后台持续生成内心独白
   - 每3-6秒生成一次
   - 真实模拟人脑的思维活动

2. **实时观察**
   - 你可以静静观察AI的思维过程
   - 看到AI如何联想、回忆、思考
   - 像观察一个"活着"的大脑

3. **交互式对话**
   - 随时输入消息打断独白
   - AI会响应你的输入
   - 回复后继续后台思维流

4. **思维状态**
   - 专注、漂移、反思、休息
   - 自然的思维跳跃
   - 情绪状态影响

## 🚀 使用方法

```bash
# 方式1：双击运行
```

## 💬 交互方式

### 观察模式
- 不输入任何内容，静静观察独白流
- 看AI如何自发思考和联想
- 每隔3-6秒更新一次

### 对话模式
- 随时输入消息
- AI会暂停独白，响应用户
- 回复后继续后台思维流

### 退出
- 输入 `quit` 或 `exit`
- 按 `Ctrl+C`

## 🎨 独白生成机制

### 思维状态机

```
专注模式 (FOCUSED)
  └─ 围绕主题深入思考
  └─ 分析用户输入
  └─ 主动思考

漂移模式 (WANDERING)
  └─ 自由联想
  └─ 记忆关联
  └─ 思维跳跃

反思模式 (REFLECTING)
  └─ 自我审视
  └─ 回顾想法
  └─ 元认知

休息模式 (RESTING)
  └─ 后台处理
  └─ 沉默思考
  └─ 记忆整理
```

### 状态转换

```python
# 专注模式
FOCUSED -> WANDERING: 15%   # 思维漂移
FOCUSED -> REFLECTING: 10%  # 自我反思

# 漂移模式
WANDERING -> FOCUSED: 30%   # 回归主题
WANDERING -> REFLECTING: 15%

# 反思模式
REFLECTING -> FOCUSED: 40%  # 重新聚焦
REFLECTING -> WANDERING: 10%
REFLECTING -> RESTING: 20%  # 休息

# 休息模式
RESTING -> FOCUSED: 50%     # 恢复专注
RESTING -> WANDERING: 30%
```

## 🐛 故障排除

### Q: 独白不生成？
**A:**
- 检查AI接口是否支持 `think()` 或 `_generate_spontaneous_monologue()`
- 查看控制台错误信息
- 确认模型已正确加载

### Q: 独白内容重复？
**A:**
- 这是正常的，人脑也会重复思考
- 如果过于频繁，可以调整温度参数
- 等待几秒，AI会切换思维状态

### Q: 输入没有响应？
**A:**
- 确保输入后按了回车键
- 检查是否在正确的窗口
- 尝试按 `Ctrl+C` 重启

### Q: 想退出但不知道怎么做？
**A:**
- 输入 `quit` 或 `exit`
- 按 `Ctrl+C`
- 关闭窗口

## 📝 参数调整

### 独白间隔

编辑 `continuous_chat.py`:

```python
# 默认: 3-6秒
self.monologue_interval = random.uniform(3, 6)

# 更频繁: 1-3秒
self.monologue_interval = random.uniform(1, 3)

# 更稀疏: 5-10秒
self.monologue_interval = random.uniform(5, 10)
```

### 独白长度

```python
# 默认: 25 tokens
return self.ai._generate_spontaneous_monologue(max_tokens=25, temperature=0.85)

# 更长: 40 tokens
return self.ai._generate_spontaneous_monologue(max_tokens=40, temperature=0.85)

# 更短: 15 tokens
return self.ai._generate_spontaneous_monologue(max_tokens=15, temperature=0.85)
```
